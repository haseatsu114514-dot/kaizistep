# ⚪︎⚪︎開示ステップメーカー

10.6秒の動画に、固定タイミングのテロップを入れて保存できる静的Webアプリです。

## 使い方

- スマホでは横画面推奨です。縦画面では案内だけを表示します。
- ブラウザの上下バーを避けたい場合は、スマホのホーム画面に追加してから開くと単体アプリ表示に近くなります。
- 左側の動画を見ながら、右側でテロップ文言を編集します。
- 通常テロップは固定時間です。
- ラスト配置テロップは7.4秒〜10.6秒に固定配置で5個表示します。
- `カメラロール保存` で保存用動画を準備し、準備後にもう一度押すとスマホの保存メニューを開きます。
- 保存メニューが使えない端末では通常ダウンロードに逃がさず、エラー表示にします。
- Web版ではOSが実際の保存先を返さないため、カメラロール以外を選んだかどうかの自動判定はできません。
