# ⚪︎⚪︎開示ステップメーカー

10.5秒の動画に、固定タイミングのテロップを入れて保存できる静的Webアプリです。

## 使い方

- スマホでは横画面推奨です。縦画面では案内だけを表示します。
- 左側の動画を見ながら、右側でテロップ文言を編集します。
- 通常テロップは固定時間です。
- ラスト配置テロップは7.4秒〜10.5秒に固定配置で5個表示します。
- 書き出し後、`ダウンロード（カメラロール保存用）` から保存します。
